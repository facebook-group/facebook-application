## Installation

 ### You must have node.js & psql installed

* Backend 

```
git clone (SSH)
```

```
npm install
```
```
add database link and port numer in .env file as follows:

DATABASE_URL=postgres://username:password@localhost:5432/db-name
PORT = port number
```
```
nodemon
```

* FrontEnd

```
git clone (ssh)
```

```
npm install
```

```
npm start
```
